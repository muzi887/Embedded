## 一、公、母到底是什么？

连接器行业里用 **公（Male）** 和 **母（Female）** 描述「凸出」和「凹入」的配合关系，和日常插头插座是同一套逻辑。

| 称呼 | 英文 | 外形 | 怎么认 | 动作 |
|------|------|------|--------|------|
| **公头 / 公针** | Male / Pin | 一根 **金属针** 凸出来 | 能 **插进** 孔里 | **插入** |
| **母头 / 母座** | Female / Socket | 一个 **塑料孔套** 包住金属簧片 | 能 **套在** 针上 | **套接** |

**口诀**：**凸出来的是公，凹进去接公的是母。**

```
公头（针）                    母头（孔）
    |                            |
    ▼                            ▼
   ─┤                           ┌─┐
    │  针                        │ │  孔，套在针上
    │                           └─┘
```

### 和开发板上的对应关系

| 板子上的东西 | 属于 | 该用什么线去接 |
|--------------|------|----------------|
| 蓝 pill **排针**（一排金属针） | **公** | 用 **母头** 那端套上去 → **公对母** 杜邦线 |
| 面包板 **插孔** | 内部有金属簧片，等效 **母** | 用 **公头** 插进去 → **公对公** 或 **公对母** 均可 |
| OLED 模块 **插座**（4 孔排母） | **母** | 模块排针朝外时，线用 **公对母** |
| ST-Link 出线端 **排针** | **公** | 用 **公对母** 接到蓝 pill |

**常见组合命名**（买线、看教程时都会遇到）：

| 线名 | 一端 | 另一端 | 最常用场景 |
|------|------|--------|------------|
| **公对母** M-F | 针 | 孔 | 开发板排针 ↔ 模块 / 面包板 |
| **公对公** M-M | 针 | 针 | 面包板 ↔ 面包板；两根排针之间 |
| **母对母** F-F | 孔 | 孔 | 两根公头排针「桥接」（较少见） |

## 二、跳线：广义概念

**跳线（Jumper Wire）** = 用来 **跨接两个电气节点** 的短线，不限定具体形态。

入门实验里常见的 **三种跳线**：

| 类型 | 别名 | 特点 | 用在哪 |
|------|------|------|--------|
| **柔性杜邦跳线** | 杜邦线 | 软线 + 两端插头，长度 10～30 cm | 接开发板、传感器模块、ST-Link |
| **面包板 U 形跳线** | 盒装跳线 | 硬铜丝弯成 U，长度固定分档 | 面包板内部短接、接电源轨 |
| **跳线帽** | 短路帽 | 黑色/黄色塑料帽，套在两相邻排针上 | BOOT0 启动选择、配置位选通 |

```
三种跳线示意：

杜邦线：  [排针]──柔性线──[面包板孔]
U 形线：  [孔]⌐
              └─[孔]     （同一侧短距离）
跳线帽：  [针][针] 上套一个帽 → 两针导通
```

**记忆**：日常说「拿几根跳线」时，在面包板场景里 **多半就是指杜邦线**；在板子角落说的「跳线」常指 **跳线帽**。

---

## 三、杜邦线（DuPont Wire）

名字来自早期 **DuPont（杜邦）** 连接器规格，现已成为通用叫法，指 **两端带 2.54 mm 间距插头** 的彩色软线。

### 3.1 规格要点

| 项 | 说明 |
|----|------|
| 间距 | **2.54 mm**（0.1 inch），与排针、面包板孔距一致 |
| 线色 | **无统一电气标准**；建议自己记「红=VCC、黑=GND、其他=信号」 |
| 长度 | 常见 10 / 20 / 30 cm；过长易缠绕、增加干扰 |
| 电流 | 只适合 **信号和小电流**；电机、大电流蜂鸣器勿指望杜邦线供电 |

### 3.2 入门套件怎么选

| 优先购买 | 原因 |
|----------|------|
| **公对母** 一盒 | 接蓝 pill 排针、ST-Link、带排母的模块 |
| **公对公** 一盒 | 面包板内部、模块间互连 |
| 母对母 | 可后补；入门阶段用得少 |

### 3.3 使用注意

1. **断电插拔**，避免热插拔损坏 IO 或模块。
2. 排针接模块插座时，**公对母更稳**；公对公容易晃、接触不良。
3. 针脚弯折、塑料帽开裂时要 **换线**，否则会出现「程序对了但板子不工作」。
4. 杜邦线 **不能替代** 正式电源线给电机、舵机大电流供电。

## 四、面包板 U 形跳线

| 项 | 说明 |
|----|------|
| 外观 | 透明分格盒，内装红/蓝/黄/绿等 **硬铜丝**，两端剥头弯成 U |
| 作用 | 面包板 **同一区域** 内短距离连线；整齐、不易松 |
| 适用 | 电源轨引入、LED 串电阻、按键上拉、相邻 5 孔组短接 |
| 不适用 | 开发板排针到面包板（间距/高度不合适时用杜邦线） |

**与杜邦线分工**：**板内短距 → U 形线；跨板、接模块 → 杜邦线。**

---

## 五、跳线帽（Jumper Cap / Shunt）

| 项 | 说明 |
|----|------|
| 是什么 | 套在 **两根相邻 2.54 mm 排针** 上的小型塑料帽，内部金属簧片使两针 **导通** |
| 常见位置 | 蓝 pill 上 **BOOT0** 旁；部分模块的地址选择、模式选择 |
| 作用 | 硬件配置：**插上 = 接通（常逻辑 1）**，**取下 = 断开（常逻辑 0）** |
| BOOT0 典型用法 | 正常运行：帽接 **0/GND**；串口下载：改接 **1/3.3V**（以板子丝印为准） |

**与杜邦线的区别**：跳线帽是 **固定短接相邻两针**；杜邦线是 **任意两点之间的柔性连接**。

## 六、调试与通信线缆（入门必用）

### 6.1 ST-Link 调试线（SWD）

| 项 | 说明 |
|----|------|
| 作用 | 向 STM32 **下载程序、单步调试、读变量** |
| 接口 | **SWD** 四线：`SWDIO`、`SWCLK`、`GND`、`3.3V`（供电可选） |
| 物理形态 | ① ST-Link 本体 + 杜邦线；② 一体扁平排线 + 4P 端子 |
| 记忆 | **ST-Link = 写进去、查得到** |

| ST-Link | 蓝 pill（常见丝印） |
|---------|---------------------|
| SWDIO | SWDIO（PA13） |
| SWCLK | SWCLK（PA14） |
| GND | GND |
| 3.3V | 3.3V（可选） |

### 6.2 USB-TTL 串口线

| 项 | 说明 |
|----|------|
| 作用 | 电脑 USB ↔ MCU **UART**，用于 `printf` 日志、AT 指令 |
| 引脚 | `VCC`、`GND`、`TXD`、`RXD` |
| 交叉规则 | **TXD → MCU RX**，**RXD → MCU TX** |
| 电平 | 选 **3.3V** 与 STM32 匹配；5V TTL 可能损坏 IO |
| 记忆 | **USB-TTL = 看串口**；**不能替代 ST-Link 烧录** |

### 6.3 Micro USB / USB Type-C 线

| 类型 | 在本项目中的用途 |
|------|------------------|
| **Micro USB** | 蓝 pill **供电**（多数板子不作 USB 设备通信） |
| **USB-A 转 Micro / Type-C** | 给开发板、串口模块、ST-Link 供电 |
| **数据线 vs 充电线** | 部分廉价线 **只有电源没有 D+/D-**，ST-Link/串口模块可能无法识别；需 **能传数据** 的 USB 线 |

## 七、模块与传感器常见接口线

入门套件和未来扩展中，模块底部或侧面常见以下接法：

| 接口形式 | 说明 | 接线方式 |
|----------|------|----------|
| **4P 排针 + 杜邦线** | OLED、蜂鸣器、传感器小板最常见 | 公对母 4 根：VCC、GND、信号 |
| **3P 排针** | 简易数字传感器（VCC/GND/OUT） | 3 根杜邦线 |
| **PH2.0 / XH2.54 端子线** | 工厂预制的插头线，扣在模块插座上 | 比杜邦更牢；买模块时注意 **间距 2.0 还是 2.54** |
| **螺丝端子** | 电机驱动、继电器、大电流电源 | 拧螺丝压线；现场设备更常见 |

**I2C 四线记忆**（OLED 等）：

| 线 | 接 MCU |
|----|--------|
| GND | GND（必须共地） |
| VCC | 3.3V |
| SCL | PB6（I2C1 常见） |
| SDA | PB7（I2C1 常见） |

## 八、后期可能用到的线材与连接器

按学习路线 **由近到远** 排列，遇到时再查本节即可。

### 8.1 排针 / 排母（Pin Header）

| 项 | 说明 |
|----|------|
| 是什么 | PCB 上焊接的 **公针排** 或 **母座排**，间距通常 2.54 mm |
| 用途 | 自己焊模块、加长排针、把公头改成母座 |
| 记忆 | 焊在板上的 **针 = 公**；**孔座 = 母** |

### 8.2 导线（飞线 / 漆包线 / 硅胶线）

| 类型 | 特点 | 用途 |
|------|------|------|
| **单芯硬线** | 弯折后成形，适合面包板 | 原型验证 |
| **多芯软线** | 耐弯折 | 机壳内部、反复插拔 |
| **漆包线** | 绝缘皮极薄 | 变压器、电机绕组（进阶） |
| **硅胶线** | 耐温、柔软 | 大电流电池、航模（进阶） |

焊接实验时需配合 **焊锡丝**；面包板阶段 **以杜邦线为主，暂不必须焊接**。

### 8.3 鳄鱼夹 / 测试钩线

| 项 | 说明 |
|----|------|
| 是什么 | 线端带金属夹，可临时夹住引脚或裸露铜线 |
| 用途 | 万用表测电压、临时抓波形、快速试通断 |
| 注意 | 仅 **临时测试**；正式电路改用杜邦线或焊接 |

### 8.4 同轴线与天线馈线（无线阶段）

| 项 | 说明 |
|----|------|
| 是什么 | 中心导体 + 屏蔽层，用于 **射频** 信号 |
| 何时遇到 | ESP8266/蓝牙天线座、LoRa 模块、外置天线 |
| 记忆 | 普通杜邦线 **不适合** 传射频；天线有专用接头（IPEX、SMA 等） |

### 8.5 网线（RJ45）与 RS-485

| 项 | 说明 |
|----|------|
| 网线 | 工业网关、以太网 MCU；**8 芯双绞**，接 RJ45 水晶头 |
| RS-485 | 现场总线 Modbus；常用 **双绞线 + 终端电阻**，非普通杜邦线长距离传输 |
| 对应学习 | [STM32嵌入式学习路线与能力规划.md](./STM32嵌入式学习路线与能力规划.md) 后期 Modbus / 联网 |

### 8.6 电源相关线（电机、舵机、电池）

| 线材 / 器件 | 说明 |
|-------------|------|
| **杜邦线** | 只传 **GPIO 信号** 或小电流 |
| **粗杜邦 / 加粗线** | 小舵机短时试用；长时间仍建议专用电源线 |
| **DC 桶头线** | 5V/12V 适配器给驱动板供电 |
| **XT30 / XT60** | 大电流锂电池接口（航模、机器人，进阶） |
| **USB 供电** | 开发调试够用；挂多个电机时电流可能不足 |

**原则**：**信号线（细）和电源线（粗）分开**；电机/舵机 **大电流回路不要只走杜邦线**。

### 8.7 FFC / FPC 软排线

| 项 | 说明 |
|----|------|
| 是什么 | 极薄的扁平柔性电缆，带锁扣插座 |
| 何时遇到 | 带触摸屏的模组、部分商用显示屏 |
| 记忆 | 插拔要看清 **0.5 mm / 1.0 mm 间距** 和 **引脚数**，弄错会毁座子 |

### 8.8 扎带、热缩管、理线扣

不算「信号线」，但实验台必备：

| 物品 | 作用 |
|------|------|
| 扎带 | 捆束杜邦线，防止扯松排针 |
| 热缩管 | 焊接后绝缘裸露焊点 |
| 魔术贴理线带 | 重复拆装比扎带方便 |

## 十、接线习惯（减少踩坑）

1. **先共地**：MCU、ST-Link、USB-TTL、OLED、传感器的 **GND 必须连在一起**。
2. **先断电再插线**：尤其 3.3V / 5V 电源脚不要带电瞎插。
3. **记颜色表**：红电源、黑地、绿 SDA、黄 SCL 等——**自己固定一套**，全项目统一。
4. **分清两种 USB 角色**：ST-Link 管烧录；USB-TTL 管串口；板子 Micro USB 多数只供电。
5. **电机大电流走专用电源**：杜邦线只接 **IN 逻辑脚**，电机电源从驱动板 VM 进。
6. **线短比线长好**：实验阶段尽量 **20 cm 以内**，减少干扰和松动。

## 十一、常见问题

| 现象 | 可能原因 | 处理 |
|------|----------|------|
| 板子时好时坏 | 杜邦线接触不良、针弯了 | 换线；排针用 **公对母** |
| 烧录失败 | SWD 线接错、BOOT0 不对 | 核对 SWDIO/SWCLK/GND；检查跳线帽 |
| 串口乱码 / 无输出 | TX/RX 没交叉、波特率不对 | TX↔RX 交叉；检查 115200 等配置 |
| 模块不亮 | VCC/GND 反了或没共地 | 断电检查；红接 3.3V、黑接 GND |
| 电机转但 MCU 复位 | 大电流从杜邦线取电 | 电机电源独立供电，只共 GND |

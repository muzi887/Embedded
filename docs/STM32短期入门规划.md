## 一、一句话理解

本规划面向 **已学过 C 语言**、跟 **江协科技（江科大）STM32 教程** 的学习者：用 **蓝 pill + ST-Link + Keil MDK + 标准外设库（SPL）**，在 **14 天、每天约 2 小时**（合计 **~28 小时**）内，掌握四个核心模块——**时钟（晶振/RCC）、GPIO、串口（USART）、中断（EXTI/NVIC）**。

```text
短期目标（2 周末）
  会配时钟 → 会 GPIO 输入输出 → 会串口 printf → 会用外部中断
  → 做出「按键中断 + 串口日志 + LED/蜂鸣器反馈」综合 Demo

长期接续
  完成本规划后 → 进入 [STM32嵌入式学习路线与能力规划.md](./STM32嵌入式学习路线与能力规划.md)
  （I2C/OLED、定时器、ADC 等按长期路线或江协后续章节自学）
```

**与长期规划的关系**：


| 对比项 | 本短期规划                           | 长期 16 周规划          |
| --- | ------------------------------- | ------------------ |
| 周期  | **2 周**                         | 16 周               |
| 工具链 | **Keil + SPL**（对齐江协视频）          | HAL / Cube 均可      |
| 终点  | 时钟 + GPIO + UART + 中断，完成入门 Demo | MQTT 上云 + 平台联调（L2） |
| 前置  | **已会 C 语言**                     | 同上                 |

### 3.1 四大模块 · 学完后你能做什么


| 模块             | 能力                                            | 验收                                        |
| -------------- | --------------------------------------------- | ----------------------------------------- |
| **时钟（晶振/RCC）** | 说清 HSE→PLL→72 MHz；会开 `RCC_APBxPeriphClockCmd` | 能解释为何 GPIO 前要开时钟；`Delay` 与 SysTick 和主频的关系 |

### 3.2 综合 Demo 验收（D14）

**「串口监控 + 按键中断」**（不接 OLED）：

```text
上电：串口打印 "STM32 Ready"，LED 慢闪（主循环或定时）

按键按下（外部中断，下降沿）：
  - LED 状态翻转
  - 串口打印 "KEY IRQ"
  - （可选）蜂鸣器短鸣一声

主循环：可做流水灯等 GPIO 演示，与中断并存
```

### 3.3 本阶段**不**覆盖（留给江协后续集或长期规划）

- I2C / OLED 显示  
- 定时器 PWM、输入捕获  
- ADC / 传感器  
- Modbus、Wi-Fi、MQTT、FreeRTOS

## 六、知识清单（四大模块）

### 6.1 时钟（晶振 / RCC）


| 概念                           | 一句话                                    |
| ---------------------------- | -------------------------------------- |
| **HSE**                      | 板载 **8 MHz 晶振**（高速外部时钟），系统主时钟常用来源      |
| **HSI**                      | 芯片内部 8 MHz RC，精度差，上电默认                 |
| **PLL**                      | 倍频；F103 常见 **8 MHz × 9 = 72 MHz** 系统时钟 |
| **SYSCLK / HCLK**            | 内核时钟；GPIO 在 APB2 上，需 **单独开外设时钟**       |
| `**RCC_APB2PeriphClockCmd`** | 使用 GPIO/USART1 等前必须调用（江协每个实验第一步）       |
| `**SystemInit()**`           | 启动文件调用，在 `system_stm32f10x.c` 里配好时钟    |
| **SysTick**                  | 内核节拍；本工程 `Delay.c` 用其做微秒/毫秒延时          |


**必会操作**：能指出蓝 pill 上 **8 MHz 晶振** 位置；能说出 **72 MHz 从哪来**；新外设知道查 **挂在 APB1 还是 APB2**。

### 6.2 GPIO


| 概念       | 一句话                                                            |
| -------- | -------------------------------------------------------------- |
| **推挽输出** | 驱动 LED、蜂鸣器；`GPIO_Mode_Out_PP`                                  |
| **上拉输入** | 读按键；`GPIO_Mode_IPU`                                            |
| **位操作**  | `GPIO_SetBits` / `ResetBits` / `WriteBit` / `ReadInputDataBit` |
| **PC13** | 板载 LED（低电平点亮，视板子电路而定）                                          |


详见 [STM32-GPIO说明.md](./STM32-GPIO说明.md)。

### 6.3 串口 USART


| 概念           | 一句话                                   |
| ------------ | ------------------------------------- |
| **USART1**   | 常用 **PA9=TX、PA10=RX**；APB2 外设         |
| **TTL 电平**   | 3.3V；USB-TTL 选 3.3V 档                 |
| **交叉接线**     | **MCU TX → 模块 RX**，**MCU RX → 模块 TX** |
| **8N1**      | 8 数据位、无校验、1 停止位；波特率 **115200**        |
| `**printf`** | 重定向 `fputc` 到 `USART_SendData`        |


详见 [MCU与UART说明.md](../DeviceAccess/Modbus/MCU与UART说明.md) §四。

### 6.4 中断（EXTI / NVIC）


| 概念             | 一句话                            |
| -------------- | ------------------------------ |
| **EXTI**       | 外部中断线；按键常接 **下降沿**             |
| **NVIC**       | 中断优先级与使能                       |
| **IRQHandler** | 如 `EXTI15_10_IRQHandler`；内清标志位 |
| **与轮询区别**      | 事件驱动，主循环可干别的事                  |


**流程记忆**：`RCC` → `GPIO` → `AFIO`/`EXTI` 线 → `NVIC` → 写 `IRQHandler`。

### 6.5 工具概念


| 概念      | 一句话                               |
| ------- | --------------------------------- |
| **SPL** | 标准外设库；`stm32f10x_gpio.c` 等        |
| **SWD** | ST-Link 烧录；**不是**串口               |
| **共地**  | MCU、ST-Link、USB-TTL、模块 **GND 相连** |

## 八、常见卡点（短期班）


| 问题                 | 快速处理                                                      |
| ------------------ | --------------------------------------------------------- |
| ST-Link 连不上        | 换 USB 口；查 SWD 四线；[环境备忘](./STM32入门环境与工程配置备忘.md)            |
| 编译报错 `stm32f10x.h` | 检查 Include 路径、Define `STM32F10X_MD, USE_STDPERIPH_DRIVER` |
| LED 不亮             | 是否开了 `RCC_APB2Periph_GPIOC`；PC13 高/低电平点亮方向                |
| 流水灯乱               | 是否开了 **GPIOA** 时钟；是否误用 `GPIO_Write` 整端口                   |
| 串口无输出              | TX/RX 交叉；波特率 115200；USB-TTL 是否 3.3V                       |
| 串口乱码               | 主频不是 72 MHz 却按 72 MHz 算波特率；查 `SystemInit`                 |
| 中断不进               | `NVIC` 是否使能；EXTI 线是否与引脚对应；是否清中断标志                         |
| 按键抖多次触发            | 中断里加简单消抖或只置标志位，主循环处理                                      |
| 越学越懵               | **停视频，回 D2 重做 LED**，再往后                                   |

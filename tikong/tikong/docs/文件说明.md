## 架构速记（避免按旧路径找文件）

| 旧习惯路径 | 当前位置 |
| --- | --- |
| `HARDWARE/*.c` 业务 | `app/`、`app/utils/` |
| `HARDWARE` 内 `*Process` / 串口业务 | `app/comm/` |
| `SYSTEM/utils/cmd.c` | `app/utils/cmd.c` |
| 片外 I2C/EEPROM/RTC 芯片驱动 | `drivers/`；时间统一经 `board/RTC.c`（`RtcChip_*`） |
| cJSON / SHA1 | `third_party/` |
| `QRProcess()` 单入口 | `QRProcessUart4()` + `QRProcessUart5()` |
| `DoCmd1–6` | `CommContrl` → `Cmd_Setting` / `Cmd_Permission` 等 |

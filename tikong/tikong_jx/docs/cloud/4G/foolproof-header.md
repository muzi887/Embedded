> **相关**：[pdf-reading-order.md](./pdf-reading-order.md)、`WH-LTE-7S1-CT-N40_说明书V1.0.0.pdf` / 硬件设计手册

## 一、一句话

**防呆（poka-yoke）**：用结构把错误插接「做不成」，而不是靠人记住方向。

插针模组往母座/底板上插时，若正反、错位都能硬塞进去，容易把电源脚接到信号脚，烧板或烧模组。防呆就是从机械上堵掉这种插法。

---

## 二、说明书那句话在说什么

模块开放接口包括：电源、复位、恢复出厂、工作状态指示、SIM、USB、UART 等，封装为 **双排插针（DIP）**。

「双排插针采用防呆设计」指的是：

- 说的是 **机械装配**（怎么插、能不能插反）；
- 目标：组装/返修时 **不易插反、不易偏一排针**。

与固件对照时：防呆保证「脚接到对的焊盘」；脚接到哪路 MCU（如 RST→PB13、UART→USART3）仍由原理图决定。

防呆」→  **装配防错**。  


---

## 三、常见防呆做法（插针模组）

具体实现以该型号外形图 / 硬件设计手册为准，常见手段包括：

| 做法 | 作用 |
| --- | --- |
| **缺针 / 堵孔** | 某一位无针或母座堵死，反插时顶住，插不进去 |
| **导槽 / 凸键（key）** | 壳体一侧有键，只能朝一个方向入座 |
| **非对称外形** | 两边形状不同，反了合不上 |
| **丝印 / 1 脚标识** | 辅助认向；单独靠丝印不算强防呆，常与机械键配合 |

WH-LTE-7S1 一类 DIP 模组：看说明书/手册里的 **引脚图、外形图、推荐焊盘**，对照 PCB 上的座子方向与 1 脚标记即可理解本板的防呆怎么对齐。

## 四、和「接口列表」的关系

说明书先列功能接口，再强调插针防呆，意思是：

```text
功能上：电源 / RST / RELOAD / LINKA… / UART / USB / SIM …
机械上：这些脚都在双排针上 → 用防呆降低插反风险
```

- **防呆**：保证物理连接方向正确；  
- **电气正确**：还要看电源电压范围、电平匹配（UART TTL、LINKA 电平说明等）——那是硬件设计手册的内容，不是「防呆」本身。

# 江协风格目录重构 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 `tikong/tikong` 外层目录对齐江协（`User/System/Hardware/Library/Start`），业务拆到 `App/Pass|Cloud|Store|Link`，第三方到 `Middlewares`；R1～R3 只搬家不改源文件名。

**Architecture:** 驱动在 `Hardware/Serial`（无 `*Process`）；业务在 `App/*`；板级器件在 `Hardware/{Storage,Time,Board,Modem}`。每阶段结束后 Keil 必须能全编译。

**Tech Stack:** STM32F407VET6、Keil MDK、SPL（`Library`）、现有 C 源码。

**设计依据:** [../specs/2026-07-17-江协风格目录重构-design.md](../specs/2026-07-17-江协风格目录重构-design.md)
